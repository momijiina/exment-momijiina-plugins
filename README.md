# exment-momijiina-plugins
Exment用のプラグインまとめ

詳細はドキュメントを見てください。

## プラグイン一覧

| 名前 | 表示名 | 種類 | パス | 概要 |
| ---- | ---- | ---- | ---- | ---- |
| AvatarDownload | アバターファイルのダウンロード | Button | button/AvatarDownload | 自分のアバターのファイルをダウンロードする |
| ShowOrganizationButton | ユーザーが所属する組織を表示 | Button | button/ShowOrganizationButton | 現在、ログインしているユーザーの組織を取得して表示します。 |
